Browser Deck

A seminar deck you can actually host on the site

The presentation keeps the slide-by-slide structure of the talk, but swaps the PDF movie annotations for native HTML5 video. That means it works in an ordinary browser, plays nicely on GitHub Pages, and can be linked like any other page on the site.

  • Every slide has a title, a short set of bullet points, and its corresponding rendered animation.
  • The deck supports keyboard navigation, clickable slide rail navigation, and deep links to individual slides.
  • The styling reuses the site's palette so it feels like part of the same place rather than an external embed.
